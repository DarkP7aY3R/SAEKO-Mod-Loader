This is an **unofficial fan-made project**. It is not affiliated with, endorsed by, or supported by **SAFE HAVN STUDIO** or **HYPER REAL**.

Back up your game files before installing mods. Humanity has survived worse, but your save folder may not.

---

## ✨ Features

- Installs the SAEKO mod loader DLL.
- Supports optional translation folders.
- If no `translation/` folder exists, the installer only installs the DLL loader, config, EXE bootstrap, and launcher.
- Includes helper tools for translation validation.

---

## 📦 Installation

1. Download or clone this repository.
2. Put your translation files in `translation/` only if you want to install translated CSV files.
3. Run:

    install.bat

If no `translation/` folder exists, the installer will still install the loader.
